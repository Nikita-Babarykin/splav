# Telegram Style Bot

Простой Telegram-бот на `Python`, который принимает готовый текст и переписывает его в стиле автора по примерам из файла.

Архитектура специально сделана легкой:
- `main.py` запускает бота
- `bot/config.py` читает настройки из `.env`
- `bot/handlers.py` принимает сообщения из Telegram
- `bot/rewrite.py` отправляет текст и примеры стиля в OpenAI
- `data/style_examples.txt` хранит примеры авторского стиля

## Что уже сделано

Бот умеет:
- принимать обычный текст сообщением;
- проверять, есть ли пользователь в списке доступа;
- отправлять текст в `OpenAI API`;
- переписывать его в стиле из файла с примерами;
- показывать `Telegram user id` через `/id`.

## 5. Создать файл `.env`

В корне проекта лежит файл `.env.example`.

Сделай так:
- в проводнике VS Code найди `.env.example`
- скопируй его
- вставь копию рядом
- переименуй копию в `.env`

Открой `.env` и подставь свои значения:

```env
TELEGRAM_BOT_TOKEN=сюда_вставь_токен_бота
OPENAI_API_KEY=сюда_вставь_openai_ключ
OPENAI_MODEL=gpt-4.1-mini
ALLOWED_USER_IDS=123456789
STYLE_EXAMPLES_PATH=data/style_examples.txt
MAX_INPUT_CHARS=12000
```

### Что означает каждая строка

- `TELEGRAM_BOT_TOKEN` — токен от `@BotFather`
- `OPENAI_API_KEY` — твой API-ключ
- `OPENAI_MODEL` — модель OpenAI
- `ALLOWED_USER_IDS` — список Telegram user id через запятую
- `STYLE_EXAMPLES_PATH` — путь к файлу с примерами стиля
- `MAX_INPUT_CHARS` — максимальная длина входного текста

## 6. Как узнать свой Telegram user id

Пока не знаешь свой `user id`, можно запустить бота, открыть его в Telegram и отправить:

```text
/id
```

Бот покажет число вида:

```text
123456789
```

Потом:
- останови бота
- впиши это число в `.env` в `ALLOWED_USER_IDS`
- снова запусти бота

Если хочешь дать доступ еще людям, просто добавь их id через запятую:

```env
ALLOWED_USER_IDS=123456789,987654321,555666777
```

## 11. Как устроено переписывание

Бот не обучает отдельную модель и не делает fine-tuning.

Он работает проще:
- берет твои примеры из `data/style_examples.txt`;
- прикладывает их к промпту;
- добавляет текст пользователя;
- просит `OpenAI` переписать текст в похожем стиле.

Это самый простой стартовый вариант. Его легко поддерживать и менять без сложной инфраструктуры.
